1-) Git é um software que é utilizado para criação de repositórios onde o usuário pode salvar mudanças em arquivos e códigos e atualizar mudanças que foram feitas por ele ou outras pessoas em outra máquina.

2-) Github é um site onde o uso do git é feito através de interface gráfica, ao invés de códigos. É a maior plataforma de compartilhamento e publicação de repositórios e é extremamente pratica, pois o site mostra quando o codigo foi modificado e quais linhas dele foram modificadas. Importante lembrar que todo repositório publicado no Github é publico, a menos que o usuario pague uma taxa.

3-) Modularização é a divisão do código, onde cada parte do código é desenvolvida em partes separadas.

4-) Não, versão do sistema é a versão na qual o software está publicada ou funcionando no atual momento, porém, podem existir diversas versões betas ou alphas do mesmo software que são utilizadas apenas por devs ou por testadores beta. Um codigo pode ser versionado diversas vezes antes de ser publicado ou colocado em uso. É possível observar o versionamento de codigo no Git, pois ele mostra as diversas modificações que foram feitas no codigo, gerando versões diferentes.
